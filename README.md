# GESTÃO DE TAREFA

Um sistema moderno de gestão de tarefas desenvolvido com **HTML, CSS e JavaScript puro**, focado em manipulação avançada da DOM, arquitetura modular e experiência de utilizador responsiva.

Este projeto foi desenvolvido com fins educativos e de demonstração de competências em desenvolvimento front-end sem frameworks.

---

## Demonstração

Acesse o projeto online:  
https://serafimelemelec.github.io/gestao-de-tarefa-js/

##  Funcionalidades

- Adicionar, editar e remover tarefas
- Marcar tarefas como concluídas
- Filtros dinâmicos (Todas / Pendentes / Concluídas)
- Sistema de prioridade (Baixa, Média, Alta)
- Pesquisa em tempo real
- Contador automático de tarefas
- Persistência de dados com `localStorage`
- Interface responsiva (mobile, tablet e desktop)
- Drag & drop (estrutura preparada para evolução)
- UI moderna com tema escuro

## Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM Manipulation)
- LocalStorage API
- Git & GitHub Pages
